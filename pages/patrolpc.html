During my internship at Patrol PC, I contributed to the development, testing, and debugging of the M3 Police Computer Control Screens. I worked with QML and C++ to blend graphical interfaces with backend hardware logic.

What I Worked On

  • Setting Search Feature — Built a searchable settings system that lets users quickly find and navigate to specific options within the Control Screens interface.
  • Application Themes — Designed and implemented multiple themes, including automatic system-matching mode, along with refreshed UI elements for a cleaner and more modern interface.
  • Dock Information Popup — Created a streamlined overlay that displays dock status, connected devices, and quick controls for field use.
  • Home Screen Widget System — Built a configurable widget framework allowing users to assign tools such as Navigation, Clock, Weather, Camera, System Health, and Gallery to the home screen.
  • QML Navigation System — Developed a fully functional in-app navigation module using QML, including rendering, map movement, routing logic, and UI components for real-time patrol use.

Additional Improvements

  • Redesigned brightness and audio controls for a more intuitive layout.
  • Improved the hotkey configuration page for smoother navigation and usability.
  • Added support for moving the icon bar between the top and bottom of the screen, improving user customization and workflow flexibility.
