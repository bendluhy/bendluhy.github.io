During my internship at Patrol PC, I contributed to the development, testing, and debugging of the M3 Police Computer Control Screens. I worked with QML and C++ to blend graphical interfaces with backend hardware logic.

Working Timeline

  1. System Familiarization & Initial Feature Testing

    Qt QML Debugging

    Onboarded to the Control Screens codebase, learned system architecture, and conducted structured feature testing on M3 hardware. Set up Qt debugging and validated bezel interactions and system behavior.

  2. Main Screen Audio & Display Controls

    QML UI Logic Debugging

    Designed, implemented, and finalized sound and brightness sliders on the main screen, ensuring stable interaction with system settings and fixing edge-case UI bugs.

  3. Dock Detection & Driver Management

    Qt Drivers USB

    Developed dock detection logic, implemented Ethernet and USB driver restart functionality, and began building dock status indicators and USB device displays.

  4. USB Device Manager (Tree System)

    Qt USB Data Structures

    Created a hierarchical USB device manager with tree-based organization, handling USB 2.0 compatibility and separating dock-connected devices from host devices.

  5. USB Manager Finalization & Update System

    QML UI Architecture Event Handling

    Finalized USB device manager with automated refresh modes, redesigned layout, and implemented a changelog system within the update menu.

  6. Event System Debugging & App Theme

    Qt Events Theming Debugging

    Diagnosed and resolved event system issues affecting barcode scanning, then began building a centralized application theming system for UI consistency.

  7. Dynamic App Theme Integration

    QML UI State Design Systems

    Extended the app theme system by adding a "follow AppTheme" option for front-page buttons, allowing dynamic UI styling across screens.

  8. Camera Module & Widget System

    Qt Modular UI Input Handling

    Started a new camera module, reworked hotkey pages, and implemented a flexible widget system to support modular UI components.

  9. Settings Search & UI Customization

    QML Search UX

    Built a settings search feature, enhanced widget behavior, fixed hotkey actions, and added support for repositioning the icon bar dynamically.

  10. Navigation & App Bar System

    QML UI Navigation Debugging

    Debugged the app bar and developed a navigation widget system to improve screen transitions and user workflow.

  11. Update Finalization & Camera Expansion

    Qt Camera Systems Integration

    Prepared the system for release updates while expanding the camera application with new features and a custom camera module.

  12. Dynamic Camera & Widget Controls

    Qt Real-Time UI State Management

    Implemented dynamic camera settings and refined widget control logic for better real-time responsiveness and configuration handling.

  13. COM / GPS Splitter Driver

    C++ Drivers Serial Communication

    Designed and began implementation of a COM port splitter driver to allow shared access to serial GPS data across multiple applications.

  • Setting Search Feature — Built a searchable settings system that lets users quickly find and navigate to specific options within the Control Screens interface.
  • Application Themes — Designed and implemented multiple themes, including automatic system-matching mode, along with refreshed UI elements for a cleaner and more modern interface.
  • Dock Information Popup — Created a streamlined overlay that displays dock status, connected devices, and quick controls for field use.
  • Home Screen Widget System — Built a configurable widget framework allowing users to assign tools such as Navigation, Clock, Weather, Camera, System Health, and Gallery to the home screen.
  • QML Navigation System — Developed a fully functional in-app navigation module using QML, including rendering, map movement, routing logic, and UI components for real-time patrol use.

Additional Improvements

  • Redesigned brightness and audio controls for a more intuitive layout.
  • Improved the hotkey configuration page for smoother navigation and usability.
  • Added support for moving the icon bar between the top and bottom of the screen, improving user customization and workflow flexibility.
