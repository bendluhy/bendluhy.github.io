Overview

During my internship at Patrol PC, I contributed to the development, testing, and debugging of the M3 Police Computer Control Screens. I worked with QML and C++ to blend graphical interfaces with backend hardware logic.

What I Worked On

  • Setting Search Feature — Built a searchable settings system that lets users quickly find and navigate to specific options within the Control Screens interface.
  • Application Themes — Designed and implemented multiple themes, including system-matching automatic theme mode, along with updated UI elements for a cleaner modern look.
  • Customizable Icon Bar — Added support for moving the icon bar between the top and bottom of the screen, improving user customization and workflow flexibility.
  • Dock Information Popup — Created an overlay that displays dock status, connected devices, and quick-access controls in a streamlined, user-friendly panel.
  • Home Screen Widget System — Developed a configurable widget framework allowing users to assign tools such as Navigation, Clock, Weather, Camera, System Health, and Gallery to the home screen.
  • QML Navigation System — Built a fully functional in-app navigation widget using QML, including rendering, map movement, routing logic, and UI components. Designed for real-time use in patrol vehicles and integrated with simulation tools during development.

Additional Improvements

  • Redesigned brightness and audio controls for a cleaner and more intuitive layout.
  • Improved the hotkey configuration page for easier navigation and usability.
