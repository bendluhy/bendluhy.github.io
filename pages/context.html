This project is currently in development. The goal is to build an automated “AI CMO” that connects to a brand’s ecommerce data—Shopify, Klaviyo, Meta Ads, and more—and produces a weekly founder-level strategy brief. Instead of dashboards and manual analysis, the system will summarize what happened, explain why key metrics changed, and recommend tactical steps a brand should take next.

The current phase focuses on architecture, planning, and building the core data and AI pipeline that will power these insights.

Planned Features

  • Backend API with FastAPI — Will handle authentication, connect to Shopify and Klaviyo, fetch data, and prepare clean metrics for analysis.
  • Rules Engine for Ecommerce Insights — Will evaluate a brand’s metrics against proven ecommerce benchmarks, detecting issues like low email revenue share, missing flows, or poor returning customer performance.
  • OpenAI-Powered Weekly Memo — The system will generate a detailed weekly memo explaining performance, highlighting trends, and giving actionable recommendations tailored to the brand’s live data.
  • Next.js Dashboard — A clean dashboard will display the weekly memo, recent metrics, and stack suggestions in a founder-friendly format.
  • Supabase Database + Auth — Will securely store user accounts, historical memos, weekly metrics, and tech stack connections.
  • Automated Weekly Cron System — The AI will run on a weekly schedule, producing new memos without the user needing to initiate anything.

Project Goals

  • Reduce the need for manual data analysis across platforms like Shopify, Klaviyo, GA4, and Meta Ads.
  • Provide small and mid-sized ecommerce teams with actionable, CMO-level recommendations.
  • Build a system that scales across brands while keeping operating costs extremely low.
